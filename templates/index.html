The SHG Assistant

The SHG Assistant is a simple, practical dashboard built to take the edge off repetitive tasks. It uses a mix of AI tools like Google Gemini, machine learning algorithms, OCR, and custom scripts to help you work faster and more efficiently.

Each automation is designed to handle a specific task — whether it's analyzing text, reading documents, or managing forms — all through an easy-to-use interface.

Tech stack: HTML, CSS, JavaScript (frontend), Flask with Python (backend), plus integrations with OpenAI, Google Gemini, OCR engines, and ML models.

How it works

This takes in the drawings and analyzes them for deficiencies. A convolutional neural network has been trained to identify areas of concern in wall assemblies and roof assemblies.
