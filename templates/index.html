Work Automation Dashboard

This dashboard is a hub for various process automations designed to streamline repetitive tasks and enhance productivity using modern tools and AI where applicable.

Each automation has its own dedicated interface and logic, some using OpenAI's APIs (like GPT and Vision) and others using traditional scripting and form handling. Everything is built using a lightweight tech stack for speed and simplicity.

Tech stack: HTML, CSS, JavaScript (frontend), and Flask (Python backend).
